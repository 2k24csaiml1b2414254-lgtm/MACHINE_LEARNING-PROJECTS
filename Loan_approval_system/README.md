# 🏦 Loan Approval ML Pipeline

## 📌 Overview

This project predicts whether a loan will be **approved or rejected** using Machine Learning. It includes data preprocessing, EDA, model training, and saving the trained model.

## 🚀 Features

* Data Cleaning & Missing Value Handling
* Encoding Categorical Features
* Exploratory Data Analysis (EDA)
* Logistic Regression Model
* Accuracy Evaluation
* Model Saving using Pickle

## 📊 Dataset

Includes features like Gender, Income, LoanAmount, Credit_History, Property_Area, and target Loan_Status.

## ⚙️ Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## 🧠 Model

Logistic Regression

## 📈 Accuracy

86%
